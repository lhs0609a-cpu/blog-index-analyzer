# 1. 사업 개요

## 1.1 기업 정보

| 항목 | 내용 |
|------|------|
| **서비스명** | Blrank (블랭크) |
| **서비스 유형** | B2C/B2B SaaS (구독형 소프트웨어) |
| **창업 형태** | 1인 창업 / 기술 스타트업 |
| **업종** | 정보통신업 > 소프트웨어 개발 및 공급업 |
| **사업 분야** | AI 마케팅 자동화, 데이터 분석, SaaS |

## 1.2 핵심 요약 (Executive Summary)

### 문제 (Problem)
- 국내 블로거 800만 명 중 95%가 월 수익 10만원 미만
- 네이버 검색 알고리즘 변화로 기존 방식의 블로그 운영 한계
- 블로그 지수, 키워드 분석에 대한 전문 지식 부재
- 네이버 검색광고 수동 관리로 인한 시간 낭비 및 비효율

### 솔루션 (Solution)
- **AI 기반 블로그 지수 분석**: 15단계 레벨 시스템으로 블로그 경쟁력 진단
- **블루오션 키워드 발굴**: 경쟁 낮고 수익성 높은 키워드 자동 추천
- **네이버 광고 자동 최적화**: AI가 15분마다 입찰가 자동 조정
- **SNS 콘텐츠 자동화**: 90일치 콘텐츠 자동 생성 및 예약 게시

### 시장 기회 (Market Opportunity)
- 국내 블로그 마케팅 시장: 연간 2.5조원 (연평균 15% 성장)
- 네이버 검색광고 시장: 연간 4.2조원
- 글로벌 마케팅 자동화 시장: 2026년 96억 달러 예상

### 수익 모델 (Revenue Model)
- 월정액 구독료: 9,900원 ~ 49,900원
- 연간 구독 할인: 20% 할인
- B2B 광고 대행 수수료: 광고비의 10~15%
- 예상 ARR (연간반복매출): 1차년도 4.1억원 → 3차년도 60억원

# 2. 창업 아이템 소개

## 2.1 서비스 개요

**Blrank(블랭크)**는 네이버 블로그 운영자와 광고주를 위한 **AI 기반 통합 성장 플랫폼**입니다.

데이터 분석 + AI 자동화 + 게이미피케이션을 결합하여, 누구나 쉽게 블로그를 성장시키고 수익화할 수 있도록 지원합니다.

## 2.2 핵심 기능

### A. 블로그 지수 분석 시스템
```
┌─────────────────────────────────────────────────────┐
│  블로그 URL 입력  →  AI 분석  →  15단계 레벨 진단   │
│                                                     │
│  📊 분석 항목:                                      │
│  - C-Rank (콘텐츠 품질 지수)                        │
│  - D.I.A. (활동 지수)                               │
│  - 포스팅 수, 이웃 수, 방문자 추이                  │
│  - 카테고리별 전문성 점수                          │
│                                                     │
│  💡 결과: "Lv.7 중급 블로거"                        │
│  → "상위 30% 수준, 3개월 내 Lv.10 도달 예상"       │
└─────────────────────────────────────────────────────┘
```

### B. 블루오션 키워드 발굴
```
┌─────────────────────────────────────────────────────┐
│  키워드 입력  →  경쟁도 분석  →  수익성 평가        │
│                                                     │
│  🏆 BOScore (Blue Ocean Score):                     │
│  - Gold: 월 예상 수익 50만원+, 경쟁 낮음           │
│  - Silver: 월 예상 수익 20-50만원                  │
│  - Bronze: 도전할 만한 키워드                      │
│                                                     │
│  📈 제공 정보:                                      │
│  - 월간 검색량                                      │
│  - 경쟁 블로그 수                                   │
│  - 예상 클릭 수익 (애드포스트 기준)                │
│  - 추천 진입 전략                                   │
└─────────────────────────────────────────────────────┘
```

### C. 네이버 광고 자동 최적화
```
┌─────────────────────────────────────────────────────┐
│  광고 계정 연결  →  AI 분석  →  자동 입찰 최적화    │
│                                                     │
│  🤖 AI 입찰 전략:                                   │
│  - Target ROAS: 목표 광고수익률 달성               │
│  - Target Position: 원하는 순위 유지               │
│  - Maximize Conversions: 전환 최대화               │
│                                                     │
│  ⚡ 자동화 기능:                                    │
│  - 15분마다 입찰가 자동 조정                        │
│  - 시간대별 예산 분배                              │
│  - 이상징후 실시간 감지 및 알림                    │
│  - 크리에이티브 피로도 분석                        │
└─────────────────────────────────────────────────────┘
```

### D. SNS 콘텐츠 자동화 (X/Threads)
```
┌─────────────────────────────────────────────────────┐
│  브랜드 정보 입력  →  AI 콘텐츠 생성  →  자동 게시  │
│                                                     │
│  📝 90일 콘텐츠 플랜 자동 생성:                     │
│  - 4:3:2:1 법칙 적용 (유용:재미:영감:홍보)         │
│  - 브랜드 페르소나 반영                            │
│  - 최적 게시 시간 자동 선정                        │
│                                                     │
│  🎯 지원 플랫폼:                                    │
│  - X (Twitter)                                      │
│  - Threads                                          │
│  - (향후) Instagram, Facebook                      │
└─────────────────────────────────────────────────────┘
```

### E. 30일 챌린지 (게이미피케이션)
```
┌─────────────────────────────────────────────────────┐
│  챌린지 참여  →  일일 미션 수행  →  배지 획득       │
│                                                     │
│  🎮 게이미피케이션 요소:                            │
│  - 일일 미션: 블로그 분석, 키워드 검색, 포스팅     │
│  - XP 경험치: 활동별 포인트 적립                   │
│  - 레벨 시스템: 1~15레벨 (입문→레전드)            │
│  - 리더보드: 커뮤니티 내 순위 경쟁                │
│                                                     │
│  📈 효과:                                           │
│  - 30일 완주율 78% (업계 평균 23%)                 │
│  - 평균 블로그 지수 2.3레벨 상승                   │
└─────────────────────────────────────────────────────┘
```

## 3.3 타겟 고객

### 1차 타겟: 전문 블로거 (B2C)
- **규모**: 약 80만 명 (활발한 수익형 블로거)
- **특징**: 월 1회 이상 포스팅, 애드포스트 수익화 목표
- **Pain Point**: 키워드 선정 어려움, 지수 관리 방법 모름
- **지불 의향**: 월 1~2만원

### 2차 타겟: 소상공인/자영업자 (B2B)
- **규모**: 약 300만 사업자
- **특징**: 네이버 검색광고 집행, 블로그 마케팅 병행
- **Pain Point**: 광고비 대비 효율 낮음, 관리 시간 부족
- **지불 의향**: 월 3~5만원

### 3차 타겟: 마케팅 대행사 (B2B)
- **규모**: 약 5,000개 대행사
- **특징**: 다수 클라이언트 광고 관리
- **Pain Point**: 수동 입찰 관리의 비효율
- **지불 의향**: 월 10만원 이상 (기업용)

## 3.4 경쟁 분석

| 경쟁사 | 주요 기능 | 가격 | 한계점 |
|--------|----------|------|--------|
| **블로그차트** | 블로그 순위 조회 | 무료 | 분석만 제공, 액션 없음 |
| **키워드마스터** | 키워드 검색량 조회 | 무료/유료 | 블로그 지수 미지원 |
| **애드버즈** | 네이버 광고 관리 | 월 5만원+ | 블로그 연계 없음 |
| **Blrank (우리)** | **통합 플랫폼** | 월 9,900원~ | - |

### 경쟁 우위
1. **통합 플랫폼**: 블로그 + 광고 + SNS 한 곳에서 관리
2. **AI 자동화**: 수동 작업 최소화
3. **게이미피케이션**: 높은 사용자 리텐션
4. **가격 경쟁력**: 경쟁사 대비 50% 저렴
