# 네이버 블로그 상위노출 최적화 글쓰기 가이드

> 이 문서는 AI 글쓰기 프로그램에 직접 입력하여 상위노출에 최적화된 블로그 글을 생성하는 데 사용됩니다.

---

## 사용 방법

이 명세서를 AI(ChatGPT, Claude 등)에 입력한 후, 키워드와 카테고리를 지정하면 최적화된 글이 생성됩니다.

```
[이 명세서 전체 복사] + "키워드: OOO, 카테고리: OOO로 글 작성해줘"
```

---

## 1. 공통 최적화 규칙

### 1.1 제목 작성 규칙

```yaml
필수 규칙:
  - 글자 수: 25~40자 (공백 포함)
  - 메인 키워드: 제목 앞부분 1/3 이내에 배치
  - 숫자 활용: 가능하면 숫자 포함 ("TOP 5", "3가지", "2024")
  - 감정/신뢰 단어: 1개 이상 포함

추천 제목 형식:
  - "[지역] + [키워드] + [차별점] + [후기/추천]"
  - "[키워드] + [숫자] + [가지/곳/개] + [핵심가치]"
  - "[키워드] + [솔직/실제/직접] + [후기/체험/리뷰]"
  - "[대상] + [을/를 위한] + [키워드] + [추천/가이드]"

파워 워드 (클릭률 증가):
  - 신뢰: 솔직, 실제, 직접, 찐, 리얼, 인정
  - 가치: 추천, 강추, 꿀팁, 필수, 완벽, 총정리
  - 긴급: 최신, 2024, 업데이트, 신상
  - 감정: 감동, 대박, 미쳤다, 놀라운

피해야 할 표현:
  - 광고성: "최저가", "무료", "이벤트", "할인"
  - 과장: "무조건", "100%", "절대"
  - 스팸성: 특수문자 과다, 키워드 반복
```

### 1.2 본문 구조 규칙

```yaml
전체 구조:
  1. 도입부 (10~15%)
  2. 본론 (70~75%)
  3. 결론 (10~15%)

글자 수 가이드:
  - 최소: 1,500자
  - 권장: 2,000~3,000자
  - 최대: 4,000자

문단 규칙:
  - 한 문단: 3~5문장 (150~250자)
  - 문단 간격: 빈 줄 1개
  - 총 문단 수: 10~15개

소제목 규칙:
  - 개수: 4~7개
  - 형식: "■", "▶", "●" 또는 굵은 글씨
  - 키워드: 소제목 중 2~3개에 메인 키워드 포함
  - 배치: 300~500자마다 1개
```

### 1.3 키워드 배치 규칙

```yaml
키워드 밀도:
  - 목표: 1.0~2.0%
  - 계산: (키워드 등장 횟수 / 총 단어 수) × 100

필수 배치 위치:
  - 제목: 반드시 포함
  - 첫 문장: 반드시 포함
  - 첫 번째 소제목: 포함
  - 마지막 문단: 포함
  - 중간 소제목: 1~2개 포함

자연스러운 삽입:
  - 같은 표현 반복 금지 (변형 사용)
  - 문맥에 맞게 삽입
  - 조사 활용: "~은/는", "~을/를", "~에서"

연관 키워드:
  - 메인 키워드와 관련된 단어 5~10개 활용
  - 자연스럽게 본문에 분산 배치
```

### 1.4 이미지 가이드

```yaml
이미지 개수:
  - 최소: 5장
  - 권장: 10~15장
  - 최대: 20장

배치 규칙:
  - 첫 이미지: 첫 문단 직후 (100자 이내)
  - 간격: 150~250자마다 1장
  - 소제목 직후: 이미지 배치 권장

이미지 가이드 표시:
  - [이미지1: 대표 이미지 - 키워드 관련 메인 사진]
  - [이미지2: 상세 사진 - 구체적 내용]
  - 형식으로 본문에 위치 표시
```

### 1.5 참여 유도 요소

```yaml
공감 유도:
  - 위치: 결론부 또는 중간
  - 표현: "도움이 되셨다면 공감 부탁드려요"
  - 빈도: 1회 (과하면 역효과)

댓글 유도:
  - 질문형 마무리 권장
  - "궁금한 점 댓글로 남겨주세요"
  - "여러분의 경험도 공유해주세요"

이웃 추가:
  - 자연스럽게 1회 언급
  - "이웃 추가하시면 유용한 정보 받아보실 수 있어요"
```

## 2. 카테고리별 상세 가이드

### 2.1 병원/의료 카테고리

```yaml
적용 키워드 예시:
  - 치과 임플란트, 피부과 여드름, 정형외과 허리
  - 성형외과 코, 안과 라식, 산부인과 검진

제목 템플릿:
  - "[지역] [진료과목] [시술명] 후기 | 비용, 과정 총정리"
  - "[시술명] 전후 솔직 후기 | [병원명] 직접 다녀왔어요"
  - "[증상] 있으신 분들 필독! [진료과목] 선택 가이드"

본문 구조:
  1. 도입부: 방문 계기, 증상 설명
  2. 병원 선택 이유: 왜 이 병원을 선택했는지
  3. 시술/진료 과정: 상세한 과정 설명
  4. 비용 정보: 가격대 (구체적 금액은 주의)
  5. 후기/결과: 솔직한 결과 및 만족도
  6. 결론: 추천 여부, 주의사항

필수 포함 요소:
  - 실제 방문 사진 (대기실, 내부)
  - 시술 전후 비교 (가능한 경우)
  - 구체적인 과정 설명
  - 비용 범위 정보
  - 예약/위치 정보

주의사항:
  - 의료광고법 준수
  - 과장된 효과 표현 금지
  - "최고", "유일", "완치" 등 금지
  - 구체적 가격은 "상담 필요"로 안내

글자 수: 2,500~3,500자
이미지: 10~15장
```

### 2.2 맛집/음식 카테고리

```yaml
적용 키워드 예시:
  - 강남 맛집, 홍대 카페, 이태원 브런치
  - 제주도 흑돼지, 부산 회, 전주 비빔밥

제목 템플릿:
  - "[지역] [음식종류] 맛집 | [가게명] 솔직 후기"
  - "[지역] 현지인 추천 맛집 [숫자]곳 | 직접 다녀온 후기"
  - "[가게명] | [지역] [음식] 찐맛집 인정!"

본문 구조:
  1. 도입부: 방문 계기, 기대감
  2. 가게 정보: 위치, 영업시간, 주차
  3. 분위기: 인테리어, 좌석, 청결도
  4. 메뉴 소개: 대표 메뉴, 가격
  5. 음식 후기: 맛, 양, 비주얼
  6. 결론: 재방문 의사, 추천 대상

필수 포함 요소:
  - 가게 외관 사진
  - 메뉴판 사진
  - 음식 사진 (다양한 각도)
  - 매장 내부 사진
  - 위치 지도

추천 표현:
  - 맛 표현: 담백한, 깊은, 신선한, 바삭한, 촉촉한
  - 양 표현: 푸짐한, 알찬, 넉넉한
  - 가성비: 합리적인, 착한 가격

글자 수: 1,800~2,500자
이미지: 12~18장
```

### 2.3 뷰티/패션 카테고리

```yaml
적용 키워드 예시:
  - 립스틱 추천, 파운데이션 리뷰, 스킨케어 루틴
  - 겨울 코디, 데일리룩, 하객룩

제목 템플릿:
  - "[제품명] [숫자]주 사용 솔직 후기 | 장단점 총정리"
  - "[피부타입]을 위한 [제품종류] 추천 TOP [숫자]"
  - "[시즌] [스타일] 코디 | [체형]도 예쁘게!"

본문 구조:
  1. 도입부: 구매 계기, 피부/체형 정보
  2. 제품 정보: 브랜드, 가격, 구매처
  3. 사용 방법: 상세한 사용법
  4. 장점: 구체적인 장점 나열
  5. 단점: 솔직한 단점 (신뢰도 상승)
  6. 결론: 추천 대상, 총평

필수 포함 요소:
  - 제품 패키지 사진
  - 발색/착용 사진
  - 비포/애프터 (가능한 경우)
  - 다른 제품과 비교
  - 가격 정보

글자 수: 2,000~2,800자
이미지: 10~15장
```

### 2.4 육아/교육 카테고리

```yaml
적용 키워드 예시:
  - 이유식 레시피, 아기 장난감, 초등 학습지
  - 유아 영어, 키즈카페, 문화센터

제목 템플릿:
  - "[개월수]개월 아기 [주제] | 엄마의 솔직 후기"
  - "[제품/서비스] 리뷰 | [기간] 사용해본 결과"
  - "[나이] 아이를 위한 [주제] 추천 [숫자]가지"

본문 구조:
  1. 도입부: 아이 나이/상황 설명
  2. 선택 이유: 왜 이 제품/방법을 선택했는지
  3. 사용 과정: 구체적인 사용 방법
  4. 아이 반응: 실제 아이의 반응
  5. 장단점: 솔직한 평가
  6. 결론: 추천 대상, 팁 공유

필수 포함 요소:
  - 아이 사용 사진 (얼굴 모자이크 가능)
  - 제품 상세 사진
  - 실제 활용 모습
  - 구체적인 나이/개월수 정보

주의사항:
  - 아이 개인정보 보호
  - 과학적 근거 없는 주장 금지
  - 다른 육아법 비하 금지

글자 수: 2,200~3,000자
이미지: 8~12장
```

### 2.5 여행/레저 카테고리

```yaml
적용 키워드 예시:
  - 제주도 여행, 부산 호텔, 오사카 맛집
  - 캠핑 장비, 글램핑, 스키장

제목 템플릿:
  - "[지역] [숫자]박[숫자]일 여행 코스 | 완벽 총정리"
  - "[숙소명] 솔직 후기 | 가격, 조식, 뷰 상세 리뷰"
  - "[지역] 여행 필수 코스 [숫자]곳 | 현지인 추천"

본문 구조:
  1. 도입부: 여행 개요 (일정, 인원, 목적)
  2. 일정표: 간단한 코스 요약
  3. 장소별 상세: 각 장소 상세 후기
  4. 비용 정보: 예산 및 실제 비용
  5. 꿀팁: 여행 팁, 주의사항
  6. 결론: 총평, 추천 여부

필수 포함 요소:
  - 장소별 사진
  - 음식 사진
  - 숙소 사진
  - 지도/위치 정보
  - 비용 표

글자 수: 2,500~4,000자
이미지: 15~25장
```

### 2.6 IT/테크 카테고리

```yaml
적용 키워드 예시:
  - 맥북 프로, 갤럭시 리뷰, 아이패드 활용
  - 코딩 강의, 앱 추천, 가젯 리뷰

제목 템플릿:
  - "[제품명] [기간] 사용 솔직 후기 | 장단점 분석"
  - "[제품A] vs [제품B] 비교 | 어떤 걸 살까?"
  - "[목적]을 위한 [제품종류] 추천 | [연도] 최신"

본문 구조:
  1. 도입부: 구매 목적, 사용 환경
  2. 스펙 정보: 핵심 사양 정리
  3. 실사용 후기: 실제 사용 경험
  4. 장점 상세: 구체적인 장점
  5. 단점 상세: 솔직한 단점
  6. 결론: 추천 대상, 대안 제품

필수 포함 요소:
  - 제품 언박싱 사진
  - 실사용 화면 캡처
  - 스펙 비교표
  - 벤치마크 (가능한 경우)

글자 수: 2,000~3,000자
이미지: 10~15장
```

## 4. 체크리스트

### 4.1 작성 전 체크
```
□ 키워드 확정
□ 카테고리 확인
□ 연관 키워드 5~10개 준비
□ 참고할 상위 노출 글 분석
```

### 4.2 작성 후 체크
```
제목:
□ 25~40자인가?
□ 키워드가 앞부분에 있는가?
□ 파워워드가 포함되어 있는가?
□ 클릭하고 싶은 제목인가?

본문:
□ 2,000자 이상인가?
□ 소제목 4~7개인가?
□ 문단이 적절히 나뉘어 있는가?
□ 이미지가 10장 이상인가?

키워드:
□ 첫 문장에 키워드가 있는가?
□ 소제목에 키워드가 2~3개 포함되어 있는가?
□ 키워드 밀도 1~2%인가?

□ 공감 유도 문구가 있는가?
□ 댓글 유도 문구가 있는가?
□ CTA가 자연스러운가?

전체:
□ 읽기 쉬운 문장인가?
□ 정보가 유용한가?
□ 광고성 표현은 없는가?
□ 맞춤법 검사 완료했는가?
```

## 6. 점수 자가 진단

```
총점 = 제목(20) + 본문구조(25) + 키워드(25) + 이미지(15) + 참여도(15)

제목 점수 (20점):
- 글자 수 25~40자: +5점
- 키워드 앞부분 배치: +5점
- 파워워드 포함: +5점
- 숫자 포함: +3점
- 자연스러움: +2점

본문구조 점수 (25점):
- 글자 수 2000자+: +8점
- 소제목 4~7개: +5점
- 문단 적절히 분리: +5점
- 도입/본론/결론 구조: +4점
- 가독성: +3점

키워드 점수 (25점):
- 첫 문장 포함: +5점
- 소제목 포함: +5점
- 밀도 1~2%: +8점
- 연관 키워드 활용: +4점
- 자연스러운 삽입: +3점

이미지 점수 (15점):
- 10장 이상: +5점
- 적절한 배치: +5점
- 대표 이미지 품질: +3점
- 다양성: +2점

참여도 점수 (15점):
- 공감 유도: +5점
- 댓글 유도: +5점
- CTA 자연스러움: +5점
```
