## 2. 타겟별 광고 카피

### [타겟 A] 블로그 초보자 (0~6개월)

**페인포인트 자극**
```
"열심히 쓰는데 왜 방문자가 안 늘까요?"
"블로그 시작했는데, 뭐부터 해야 할지 모르겠다면"
"3개월 써도 방문자 100명... 뭐가 문제일까요?"
"다른 사람들은 어떻게 상위 노출되는 거죠?"
```

**솔루션 제시**
```
"하루 10분, 30일이면 상위 노출 비법 마스터"
"초보도 따라하는 30일 블로그 성장 로드맵"
"AI가 알려주는 '딱 맞는' 개선 포인트"
"무료로 내 블로그 점수 확인하세요"
```

**CTA**
```
"지금 무료로 블로그 분석하기 →"
"30일 챌린지 시작하기 →"
"내 블로그 점수 확인하기 →"
```

## 3. 기능별 광고 카피

### [기능] 블로그 지수 분석

**헤드라인**
```
"내 블로그 점수, 몇 점인지 아세요?"
"상위 1% 블로거와 나의 차이, 숫자로 확인"
"40개 지표로 정확하게 분석합니다"
"3분 만에 블로그 분석 결과 받기"
```

**서브카피**
```
"C-Rank, D.I.A. 뭐가 부족한지 정확히 알려드립니다"
"11단계 레벨 시스템으로 내 위치 확인"
"AI가 맞춤형 개선 방법까지 제시"
```

---

### [기능] 블루오션 키워드

**헤드라인**
```
"아직도 경쟁 키워드에서 싸우고 계세요?"

"진입 확률 80% 이상 키워드만 추천"
```

**서브카피**
```
"BOS 점수로 키워드 등급 한눈에 확인"
"내 점수로 이 키워드 진입 가능한지 바로 확인"
"권장 글자수, 이미지 개수까지 가이드 제공"
```

---

### [기능] 순위 추적

**헤드라인**
```
"내 포스팅, 지금 몇 위인지 알고 계세요?"

"자동 순위 추적으로 성과 눈으로 확인"
```

**서브카피**
```
"키워드 자동 추출, 순위 자동 추적"

"개선 효과를 정량적으로 측정"
```

---

### [기능] 광고 자동 최적화

## 4. A/B 테스트용 변형 카피

### 테스트 1: 감정 vs 논리

**감정적 접근 (A)**
```
"더 이상 혼자 고민하지 마세요"
"블로그 성장, 답답했던 마음 이제 해결됩니다"
"드디어 찾았다, 블로그 성장 솔루션"
```

**논리적 접근 (B)**
```
"40개 지표로 정확하게 분석"
"데이터 기반 의사결정"
"숫자로 증명되는 성장"
```

---

### 테스트 2: 문제 vs 솔루션

**문제 강조 (A)**
```
"왜 아무리 써도 순위가 안 오를까요?"
"광고비만 나가고 ROI는 마이너스?"
"성장이 멈춘 느낌, 맞죠?"
```

**솔루션 강조 (B)**
```
"AI가 순위 올리는 방법 알려드립니다"
"광고비 30% 절감 자동화 시스템"
"정체기 탈출 블루오션 키워드"
```

---

### 테스트 3: 숫자 vs 감성

**숫자 강조 (A)**
```
"40개 지표, 11단계 레벨, 92% 정확도"
"방문자 3배 증가, 광고비 32% 절감"
"월 9,900원, 하루 300원"
```

**감성 강조 (B)**
```
"드디어 성장하는 내 블로그를 보세요"
"경쟁에서 벗어나 여유롭게"
"전문가처럼, 하지만 쉽게"
```
