# 슬라이드 5: 핵심 기능 1 - 블로그 지수 분석

## "내 블로그, 정확히 어느 수준인지 알려드립니다"

### 기능 설명
- 블로그 ID만 입력하면 **40개 이상의 지표** 분석
- **11단계 레벨 시스템**: Iron → Bronze → Silver → Gold → Platinum → Diamond → Challenger
- C-Rank(출처 신뢰도), D.I.A.(문서 품질) 상세 분석
- 15일간 방문자 추이 그래프
- AI 기반 **맞춤형 개선 권장사항**

### 광고 카피 예시
```
[메타 광고용]
"내 블로그 점수, 객관적으로 알고 계신가요?"
"상위 1% 블로거와 나의 차이, 숫자로 확인하세요"
"무료 블로그 분석으로 성장 전략을 찾아보세요"

[블로그 글 제목]
"네이버 블로그 점수 확인하는 법 - 무료 분석 도구 추천"
"내 블로그가 상위 노출 안 되는 진짜 이유 (점수로 확인)"
```

### 타겟 키워드
- 블로그 지수 확인
- 블로그 점수 분석
- 네이버 블로그 분석
- 블로그 C랭크 확인

# 슬라이드 6: 핵심 기능 2 - 블루오션 키워드 발굴

## "경쟁 없이 상위 노출되는 키워드를 찾아드립니다"

### 기능 설명
- **BOS(Blue Ocean Score)** 기반 키워드 등급화
  - 🥇 Gold (80+): 황금 키워드 - 즉시 진입 추천
  - 🥈 Silver (60-79): 좋은 기회
  - 🥉 Bronze (40-59): 도전 가능
  - ⚪ Iron (20-39): 경쟁 있음
  - 🔴 Blocked (0-19): 레드오션
- 상위 블로그 평균 점수 분석
- **진입 확률** % 표시
- 권장 글자 수, 이미지 개수 제시
- 구체적인 **공략 팁** 제공

### 광고 카피 예시
```
[메타 광고용]
"아직도 경쟁 키워드에서 싸우고 계신가요?"
"검색량은 높고, 경쟁은 낮은 키워드 - AI가 찾아드립니다"
"블루오션 키워드로 상위 노출 성공하세요"

[블로그 글 제목]
"블루오션 키워드 찾는 법 - AI 도구로 5분 만에"
"네이버 블로그 상위 노출 비법: 경쟁 피하는 키워드 전략"
"초보 블로거도 상위 노출되는 키워드 선정법"
```

### 타겟 키워드
- 블루오션 키워드
- 블로그 키워드 찾기
