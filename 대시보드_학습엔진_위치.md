# 📍 대시보드에서 학습 엔진 위치 안내

## ✅ 이미 구현되어 있습니다!

학습 대시보드는 **메인 대시보드 페이지**에 이미 완벽하게 구현되어 표시되고 있습니다.

---

## 📍 화면에서 보이는 위치

### 1️⃣ 상단 헤더 버튼
대시보드 상단 오른쪽에 **3개의 버튼**이 있습니다:

```
┌─────────────────────────────────────────────────┐
│ 대시보드                                         │
│ 내 블로그 지수를 한눈에 확인하세요                  │
│                                                  │
│  [AI 학습 엔진] [키워드 검색] [블로그 추가]        │
└─────────────────────────────────────────────────┘
```

- **AI 학습 엔진** 버튼 (초록색 테두리) - 클릭하면 `/dashboard/learning` 페이지로 이동

---

### 2️⃣ 키워드 지수분석 섹션 (첫 번째 섹션)

```
┌──────────────────────────────────────────────────┐
│ 📈 키워드 지수분석                      [분석 시작] │
│ 경쟁 키워드의 상위 블로그들을 분석하세요             │
│                                                   │
│ ┌─────────┐  ┌─────────┐  ┌─────────┐           │
│ │👁 상위    │  │📈 경쟁   │  │✨ 노출   │           │
│ │노출 분석  │  │인사이트  │  │로직 파악 │           │
│ └─────────┘  └─────────┘  └─────────┘           │
└──────────────────────────────────────────────────┘
```

이 섹션에서 **[분석 시작]** 버튼을 클릭하면 키워드 검색 페이지로 이동합니다.

---

### 3️⃣ 학습 대시보드 섹션 (두 번째 섹션) ⭐ 여기입니다!

```
┌──────────────────────────────────────────────────┐
│ 🧠 학습 대시보드                  [학습 엔진 보기] │
│ 순위 학습 현황을 실시간으로 모니터링하세요           │
│                                                   │
│ ┌─────────┐  ┌─────────┐  ┌─────────┐           │
│ │📈 실시간 │  │👁 가중치 │  │✨ 예측   │           │
│ │학습      │  │모니터링  │  │정확도    │           │
│ │          │  │          │  │          │           │
│ │검색할 때 │  │C-Rank,  │  │학습이    │           │
│ │마다 자동 │  │D.I.A.   │  │진행될수록│           │
│ │으로 순위 │  │가중치가  │  │순위 예측 │           │
│ │데이터를  │  │어떻게   │  │정확도가  │           │
│ │학습합니다│  │변하는지  │  │향상됩니다│           │
│ └─────────┘  └─────────┘  └─────────┘           │
└──────────────────────────────────────────────────┘
```

이 섹션에서 **[학습 엔진 보기]** 버튼을 클릭하면 학습 대시보드 페이지로 이동합니다!

---

그 아래로 블로그 검색 바와 블로그 카드들이 표시됩니다:

```

│ 🔍 블로그 검색...                                  │

│72.5점   │  │62.0점   │  │86.5점   │
└─────────┘  └─────────┘  └─────────┘
```

## 🚀 학습 대시보드 접근 방법

### 방법 1: 상단 버튼 클릭
대시보드 상단의 **[AI 학습 엔진]** 초록색 버튼 클릭

### 방법 2: 학습 대시보드 섹션 버튼 클릭
중간에 있는 학습 대시보드 섹션의 **[학습 엔진 보기]** 버튼 클릭

### 방법 3: 직접 URL 접속
- 로컬: http://localhost:3006/dashboard/learning
- 프로덕션: https://blog-index-analyzer.vercel.app/dashboard/learning

---

## 🎨 구현된 기능

### 메인 대시보드 (`/dashboard`)
- ✅ 학습 대시보드 섹션 표시
- ✅ 3개 카드 (실시간 학습, 가중치 모니터링, 예측 정확도)
- ✅ [학습 엔진 보기] 버튼
- ✅ 초록색 그라데이션 디자인

### 학습 대시보드 페이지 (`/dashboard/learning`)
- ✅ 실시간 통계 카드 4개
- ✅ 정확도 향상 라인 차트
- ✅ 가중치 분포 시각화
- ✅ 최근 학습 이력 테이블
- ✅ 수동 학습 실행 버튼
- ✅ 30초마다 자동 갱신
