# 블로그 지수 분석기 - Chrome Extension

네이버 블로그 통계를 자동으로 수집하고 분석 서버로 전송하는 Chrome Extension입니다.

## 기능

1. **자동 통계 추출**: 네이버 블로그 통계 페이지에서 방문자 수, 포스트 수, 이웃 수 자동 추출
2. **수동 입력**: 통계를 직접 입력 가능
3. **분석 서버 연동**: 추출한 데이터를 로컬 분석 서버(localhost:8001)로 전송
4. **결과 확인**: 분석 완료 후 자동으로 결과 페이지(localhost:3000)로 이동

## 설치 방법

### 1. 아이콘 생성 (필수)

Extension을 설치하기 전에 아이콘 파일이 필요합니다.

#### 옵션 A: 온라인 도구 사용
1. https://favicon.io/ 또는 https://www.favicon-generator.org/ 방문
2. 텍스트나 이모지로 아이콘 생성 (예: 📊)
3. 16x16, 48x48, 128x128 크기의 PNG 파일 다운로드
4. `E:\\u\\blog-index-analyzer\\chrome-extension\\icons\\` 폴더에 저장:
   - `icon16.png`
   - `icon48.png`
   - `icon128.png`

#### 옵션 B: 간단한 placeholder 사용
아이콘이 없으면 일단 manifest.json에서 icons 부분을 주석 처리하고 설치 가능

### 2. Chrome에 Extension 설치

1. Chrome 브라우저 열기
2. 주소창에 `chrome://extensions/` 입력
3. 우측 상단의 **개발자 모드** 활성화
4. **압축해제된 확장 프로그램을 로드합니다** 클릭
5. `E:\\u\\blog-index-analyzer\\chrome-extension` 폴더 선택
6. 설치 완료!

## 사용 방법

### 방법 1: 수동 입력 (추천)

1. Extension 아이콘 클릭
2. **수동 입력** 섹션에 블로그 통계 입력:
   - 블로그 ID: `lhs0609c`
   - 전체 포스트 수: `50`
   - 총 방문자 수: `10000`
   - 이웃 수: `30`
3. **📤 분석 서버로 전송** 버튼 클릭
4. 분석 완료 후 자동으로 결과 페이지로 이동

### 방법 2: 자동 추출 (실험적)

1. 네이버 블로그 페이지 방문
2. Extension 아이콘 클릭
3. **현재 페이지에서 통계 추출** 버튼 클릭
4. 추출된 데이터 확인
5. 필요시 수동으로 수정
6. **📤 분석 서버로 전송** 버튼 클릭

## 주의사항

- 백엔드 서버(localhost:8001)가 실행 중이어야 합니다
- 프론트엔드 서버(localhost:3000)가 실행 중이어야 결과 확인 가능
- 네이버 블로그의 통계 페이지 구조가 변경되면 자동 추출이 실패할 수 있습니다
- 그런 경우 수동 입력을 사용하세요

## 트러블슈팅

### Extension이 로드되지 않을 때
- manifest.json에 오류가 없는지 확인
- icons 폴더에 아이콘 파일이 있는지 확인 (없으면 manifest.json에서 icons 제거)

### 통계 추출이 안될 때
- 네이버 블로그 통계 페이지에 있는지 확인
- 브라우저 콘솔(F12)에서 에러 메시지 확인
- 수동 입력 사용

### 서버로 전송이 안될 때
- 백엔드 서버(localhost:8001)가 실행 중인지 확인
- CORS 설정 확인
- 브라우저 콘솔에서 네트워크 에러 확인

## 개발자 정보

- 백엔드 API: `http://localhost:8001/api/blogs/analyze`
- 프론트엔드: `http://localhost:3000/analyze`
- Content Script: 네이버 블로그 페이지에서 자동 실행
- Popup: Extension 아이콘 클릭 시 표시
