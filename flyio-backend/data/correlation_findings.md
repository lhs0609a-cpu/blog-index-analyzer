## 4라운드 종합 결론

외부에서 측정 가능한 어떤 단일 변수도 SERP 순위를 강하게 예측하지 못함. 가장 신뢰할 만한 신호는 **딱 4개**, 모두 |ρ| ≈ 0.13~0.20:

| 검증된 약한 양의 상관 신호 | ρ | 의미 |
|---|---|---|
| **post_freshness** (최근 작성일) | 0.183 | SERP는 신선한 글 선호 |
| **content** (보강된 콘텐츠 점수) | 0.153 | 길이+이미지+구조 결합 |
| **fp_images** (실제 이미지 수) | 0.148 | 이미지 풍부한 글 |
| **context** (카테고리 엔트로피) | 0.134 | 한 주제 집중 블로그 |

나머지 30+ 신호 |ρ| < 0.10 = 통계적 노이즈.

## 시사점 (정직하게)

### 1. "외부 신호로 SERP 예측"은 본질적 한계
- 네이버 D.I.A.+의 딥매칭은 의미적 검색 매칭 — 문서 raw 메트릭으로 reverse-engineer 불가능
- 블로그 분석기는 **운영 진단 도구**로 포지셔닝, SERP 예측은 별도 시스템 필요

### 2. 의미 있는 신호는 "신선도 + 콘텐츠 풍부도"
- 최신성과 콘텐츠 충실도가 약하지만 일관되게 양의 신호
- 가중치 재조정 필요: freshness/content 비중 ↑, engagement/keyword 비중 ↓

### 3. SERP 모니터링은 별도 트랙으로
- 우리 분석기로 SERP 예측 시도 ❌
- 키워드별 SERP 시계열 추적 ✅ (판다랭크 방식)
- 본인 블로그의 글이 검색에 노출되는지 누락율 측정 ✅
